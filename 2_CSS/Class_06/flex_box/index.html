flex-basis sets the initial size of an item before the remaining space is distributed. Can be a length (px, em) or auto(uses content size).

8. Flex Shorthand

The flex property combines grow, shrink and basis in one declaration. Common values: flex 1 ( grow and shrink equally), flex: 2 ( grow twice as much ), flex: initial (don't grow), flex:none(fixed size).
